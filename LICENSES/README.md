# License inventory

The root `LICENSE` contains the complete GPLv2 text governing PeonPad's
original scaffold and intended Stratagus/Wargus derivative work. Every
additional third-party or libre-asset license must be added here when the
corresponding revision-locked input is staged. A source or asset cannot enter
the build while its license is `MISSING` in `config/inputs.lock`.

The direct SDL3 foundation lane carries the upstream Zlib license texts in:

- `SDL3.txt`
- `SDL3_image.txt`
- `SDL3_mixer.txt`
