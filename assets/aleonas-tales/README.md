# Aleona's Tales asset boundary

Only a complete, revision-locked, license-verified libre Aleona's Tales asset
set may be staged here. The current `ref/data.Wargus` tree is Blizzard-derived
runtime data and must never be used to populate this directory.

The current staged snapshot is **local-test-only**. Its media audit has 797
unresolved files, is ignored at `assets/aleonas-tales/source/`, and must not be
committed, pushed, or distributed. See
`docs/aleona-asset-audit.md` and run
`scripts/audit-aleona-assets.sh --strict` before any release bundle is made.
